### 검토서 작성 내용 및 유효성 검사

샘플은 /.doc/2025-0005.xlsm 참조

## 검토서에서 내용 확인 해야될 것.
1. 관리번호, 절차
2. 검토위원
3. 검토결과
4. 건축사 소속 / 성명
5. 책임구조기술자 소속 / 성명
6. 유형별 상세 검토 (2)전이구조 우측에 "필로티"라는 단어가 추가되면 필로티로 별도 구분
7. 주요 구조형식
8. 구조도면 작성자의 자격
9. 고위험 유형
10. 적정성 검토 결과 (현재 79열에 있으나 행이 추가되면 더 아래쪽에 기입될 수 있음)
11. 판정결과 부적합 유형 1~3개

## 검토서의 유효성 검사
1. 파일명 : 관리번호.xlsm 으로만 되어야 함. (예 : 2025-0001.xlsm)
2. 관리번호 일치 여부
3. 검토위원 일치 여부
4. 적정성 검토 결과  "적합"인데 판정결과 부적합 유형 첫번째 "적합", 나머지는 빈칸이어야 함.
5. 적정성 검토 결과에 내용이 기입되어 있는데 "판정결과 부적합 유형" 이 "적합"으로 기입 또는 비어 있는 경우
6. 예비검토 접수 상태 : 절차 = "1차 적정석 검토", 탭이름 "검토서(1차)"
7. 보완검토 접수 상태 : 절차 = "2차 적정성 검토", 탭이름 "검토서(2차)" (보완 검토의 경우 1차,2차,3차 여도 무조건 절차와 탭이름은 이렇게 구성됨)
8. 탭이 2개 이상 있으면 안됨.

## 제출전 확인 사항
1. 건축사 및 책임기술자 확인
2. 유형별 상세 검토 확인
3. 주요 구조형식 확인
4. 구조도면 작성자 자격 확인
5. 고위험 유형 확인
6. 적정성 검토 결과 확인
7. 판정결과 부적합 유형 확인
